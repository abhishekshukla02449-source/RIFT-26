# Blood Donor Network

A simple Flask web app to connect blood donors with people in need.

## Problem

Finding compatible blood donors quickly during emergencies is difficult.

## Solution

Our app provides a searchable donor directory by city and blood group, making it faster to connect patients with donors.

## Features

- Donor registration (name, phone, city, blood group)
- Search donors by city and blood group
- Simple, responsive interface
- SQLite database for quick storage

## Tech Stack

- Backend: Python (Flask)
- Database: SQLite (SQLAlchemy ORM)
- Frontend: HTML, CSS (Jinja templates)

## Roadmap

- Add donor verification (OTP/email)
- Availability toggle & last donation date
- Deploy on Render/Heroku for live demo
